# 📊 Telco Customer Churn Analysis & Prediction

An end-to-end Machine Learning project that analyzes customer churn in the telecom industry and predicts whether a customer is likely to churn using classification models.

---

## 🚀 Features

- 📊 Dataset Overview
- 📈 Exploratory Data Analysis (EDA)
- 🤖 Model Performance Comparison
- 🔮 Customer Churn Prediction
- 📥 Download Dataset
- 🌐 Interactive Streamlit Dashboard

---

